# 修仙RPG设计文档

## 系统
### 操作战斗
```
想法: 方便后续手机端的移植和玩家上手门槛, 尽量简化操作, 按键, 整体参考王者荣耀, 但去掉需要选取目标的技能, 尽量自动

移动WASD(左摇杆)

普攻J(R1) 闪躲奔跑K(X) 123技能UIO(口/三角/圆) 后续往右拓展(如无意外尽量不要拓展)
```

### 人物属性
```
想法: 
    整体参考王者荣耀, 物理系对应武术, 魔法系对应法术
    加入体力, 后期能解锁无限体力, 增加成长爽感

基础:
    MaxHealth
    MaxMana
    CurHealth
    CurMana
    Level
    CurExp 经验
    ExpToNextLevel 升级所需总经验
移动:
    Stamina 体力
    MoveSpeed
战斗:
    AtkSpeed 攻速

    PhysicalAttack pAtk 物攻
    PhysicalDefense pDef 物防
    PhysicalPenetration pPen 物穿

    MagicalAttack mAtk 魔攻
    MagicalDefense mDef 魔防
    MagicalPenetration mPen 魔穿

    CritRate 暴击率(概率)
    CritDamage 暴击伤害(倍率)
特殊效果:
    LifeSteal 吸血


```
### 装备
- 装备部位, 武器+攻击, 防具+物防魔防
    - 手, 武器, 特色+吸血+暴击
        - 种类 : 剑 刀 枪 弓 
    - 头, 防具, 特色+最大法术值
    - 胸, 防具, 特色+大量+物防魔防
    - 靴, 防具, 特色+移速, 
- 等级划分
    - 报废级 白色
    - 普通级 绿色
    - 稀有级 蓝色
    - 史诗级 紫色
    - 传说级 金色
- 每件装备都有自己的等级和阶层, 满级100, 每20级破境, 破境需要材料(提升肝度, 量大管饱), 破境的提升随机
- 尽量提高随机性(增加丰富度, 同时能减少工作累, 不需要搞很多个不同的装备)
    - 武器的属性在等级区级内随机
    - 破境的提升随机
```
bd举例
73级金剑
33级紫头
45级绿胸
88级金靴
```
### 仓库(背包)
- 装备不能堆叠, 材料可以堆叠
- 有容量, 需要回家存东西, 容量可以升级
